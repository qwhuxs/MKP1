# LightHTML Patterns

Цей проект демонструє використання різних шаблонів проектування в PHP для маніпуляції з DOM-структурами за допомогою класу LightHTML. Приклади реалізації таких шаблонів:

- **Команда** 
- **Відвідувач** 
- **Ітератор** 
- **Стейт** 
- **Шаблонний метод**
